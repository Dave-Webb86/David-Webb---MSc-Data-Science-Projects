# Yelp Dataset (2021)

This folder contains a coursework investigating the [Yelp.com](https://www.yelp.com/dataset) dataset, accessed in January 2021.

Works consist of 2 tasks:
 - 3 designated questions to establish confidence querying data using the Pyspark framework
 - A self directed investigation of the data, looking at which review types are most useful for business owners
